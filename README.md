## 簡易cms（ブログ）

### アプリ概要
php・javascript共に理解を深める学習目的で、フレームワークやライブラリを用いずに作成したブログです。  
管理者側と閲覧者側に分かれていて、ログイン機能は動作させてないです（コメントアウト済み）。  
[サイトへ移動](https://dokusika.deca.jp/site/portfolio-blog/public/)

### アプリ機能
記事のCRUD機能  
画像投稿  
ページネーション  
カテゴリー機能  
コメント機能  
（認証機能）
